Editeur d'animations - Matrice LED

Etape 3/3 : previsualiser l'animation et sauvegarder le travail localement. Un octet par colonne, bit 0 = ligne du haut (meme convention que le sketch MatriceLED.ino).

Passez en Mode : Image ci-dessus puis glissez l'image (souris ou doigt) pour la deplacer. Utilisez le curseur Echelle pour zoomer.

Upload GitHub

1. Creer un token GitHub (Personal Access Token) :

  • Aller sur github.com → photo de profil → Settings
  • Tout en bas du menu de gauche : Developer settings
  • Personal access tokens → Tokens (classic) → Generate new token (classic)
  • Cocher la case repo (acces complet aux depots), donner une date d'expiration, puis Generate token
  • Copier le token affiche (il commence par ghp_) : il ne sera plus visible ensuite, collez-le dans le champ Cle ci-dessus

Erreur "403 Resource not accessible" ? C'est un probleme de permissions du token :

  • Si vous avez cree un token fine-grained : verifiez qu'il donne bien acces au depot concerne et que la permission Contents : Read and write est activee
  • Si vous avez cree un token classic : verifiez que la case repo etait bien cochee
  • Si le depot appartient a une organisation avec SSO : dans la liste des tokens, cliquez sur Configure SSO a cote du token et autorisez-le pour l'organisation
  • Le plus simple en cas de doute : regenerer un nouveau token classic avec la case repo cochee

2. Renseigner l'adresse :

  • Format : owner/repo/dossier (sans le nom de fichier)
  • Exemple : si votre depot est github.com/curtis/matrice-led et que vous voulez sauver dans le dossier animations, ecrivez curtis/matrice-led/animations
  • Le nom du fichier est genere automatiquement a partir du nom du tableau C (champ plus haut), suivi de .anim
  • Si un fichier du meme nom existe deja dans le dossier, un numero est ajoute automatiquement (monAnimation_2.anim, etc.) pour ne rien ecraser
  • Le dossier est cree automatiquement s'il n'existe pas encore

3. Branche (optionnel) : laissez vide pour utiliser la branche par defaut du depot (souvent main).

Le token n'est jamais enregistre : il faudra le recoller a chaque visite de la page.
