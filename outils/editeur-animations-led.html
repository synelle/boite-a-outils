Editeur d'animations - Matrice LED

Etape 3/3 : previsualiser l'animation et sauvegarder le travail localement. Un octet par colonne, bit 0 = ligne du haut (meme convention que le sketch MatriceLED.ino).

Upload GitHub

1. Creer un token GitHub (Personal Access Token) :

  • Aller sur github.com → photo de profil → Settings
  • Tout en bas du menu de gauche : Developer settings
  • Personal access tokens → Tokens (classic) → Generate new token (classic)
  • Cocher la case repo (acces complet aux depots), donner une date d'expiration, puis Generate token
  • Copier le token affiche (il commence par ghp_) : il ne sera plus visible ensuite, collez-le dans le champ Cle ci-dessus

2. Renseigner l'adresse :

  • Format : owner/repo/chemin/fichier.anim
  • Exemple : si votre depot est github.com/curtis/matrice-led et que vous voulez sauver le fichier dans le dossier animations, ecrivez curtis/matrice-led/animations/monAnimation.anim
  • Le dossier et le fichier sont crees automatiquement s'ils n'existent pas encore

3. Branche (optionnel) : laissez vide pour utiliser la branche par defaut du depot (souvent main).

Le token n'est jamais enregistre : il faudra le recoller a chaque visite de la page.
